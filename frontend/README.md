# equipo-c24-74-m-webapp

Instrucciones para iniciar el proyecto:

1. Clonar o forkear el repositorio:

```bash
git clone https://github.com/No-Country-simulation/equipo-c24-74-m-mobile/
cd frontend
```

2. Instalar las dependencias

```bash
npm install
```

3. Ejecución ▶️

```bash
npm run dev
```
